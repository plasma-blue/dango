Dango is a lightweight, atomic thought-organizing tool born for **efficient block diagramming**.

It doesn't try to be a complex drawing software or a heavy note-taking app. Instead, it helps you quickly organize scattered thoughts and clarify relationships the moment inspiration strikes.
